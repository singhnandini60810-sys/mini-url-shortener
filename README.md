# Mini URL Shortener

A simple web application that shortens long URLs and stores them locally in the browser.

## Features

- Shorten long URLs
- Generate unique short links
- Store URL history
- Works offline using local storage
- Simple and clean interface

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Enter a long URL
2. Click shorten
3. Get short URL
4. View history

## Future Improvements

- Copy short URL
- Open short link
- Backend integration
- API-based shortening
